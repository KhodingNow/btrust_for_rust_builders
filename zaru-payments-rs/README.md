# zaru-payments-rs

**Instant ZARU merchant payments on Solana** — built for South Africa’s retail stablecoin rollout.

One-click invoices + direct ZARU settlement. No custody. No middleman. Perfect for Luno, Lesaka, EasyEquities & Sanlam.

### Why this is important ?
- Uses official $ZARU (rand-backed, audited by Moore Johannesburg)
- Aligns 100% with Solana Developer Platform (SDP) Payments module (launched 24 Mar 2026)
- Ready for on-ramp (Luno fiat→ZARU) + off-ramp (Lesaka bank rails)
- Production-ready Rust + Anchor (tests, CI, error handling)
